This is just a command-line tool specifically a text editor tool that transforms text. You can:
- Make text uppercase or lowercase
- Capitalize words or make title case
- Convert to snake_case
- Reverse each word
- Count letters, words, spaces, and characters
- Check if a string is a palindrome
- It handles extra spaces, numbers, punctuation, and mixed-case commands.

How to run
Save the program as main.go. Open terminal in that folder and run the program using the usual "go run main.go"

Type and test commands or text. Exit function is there to quit whenever the user wants to exit. It also handles edge cases and shows history of past result.
